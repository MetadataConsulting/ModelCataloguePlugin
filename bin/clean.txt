Cleans all the resources generated by project and refreshes dependencies

This includes
 * 'build' folder in the root folder
 * generated JavaScript fixtures
 * calling 'clean all' for each Grails subproject
 * deleting 'target' and 'target-eclipse' folders for each Grails subproject
 * deleting related folders in '~/.grails' folder for each Grails subproject
 * calling 'refresh-dependencies' for each Grails subproject
